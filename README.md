# 项目说明

通过阅读 vue 源码来学习如何写一个 MVVM 框架。

# why?

1、学习 vue 的双向数据绑定原理以及核心代码模块
2、缓解好奇心的同时了解如何实现 vue

# 进展

- [x] 监听对象变化
- [x] 监听数组变化
- [x] v-on 事件绑定
- [x] v-model 双向数据绑定
- [x] 实现\$del
- [x] 实现\$set

# 如何启动

```
yarn global add live-server

live-server
```
