# 项目说明

通过阅读 vue 源码来学习如何写一个 MVVM 框架。

# why?

1、学习 vue 的双向数据绑定原理以及核心代码模块
2、缓解好奇心的同时了解如何实现 vue

# 进展

- [x] 监听对象变化
- [ ] 监听数组变化
- [ ] 计算属性 computed
- [x] watch 库
- [ ] 动态数据绑定
- [ ] v-if 条件渲染
- [ ] v-on 事件绑定
- [ ] v-model 双向数据绑定
- [ ] 生命周期(面向切面编程)
